Different Methods:
--------------------------------------------------------------

POST : https://nodejs-filesystem-v47f.onrender.com/api/files/create

Above API endpoint will create a text file in directory /opt/render/project/src/testFolder on server.
Content of the file will be current timestamp & the file name will be current date-time.txt

--------------------------------------------------------------

GET : https://nodejs-filesystem-v47f.onrender.com/api/files/

Above API endpoint will retrieve all the text files in a directory /opt/render/project/src/testFolder from server.
